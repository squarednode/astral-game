# Project Vision

Project Astral is a browser-based 2.5D action RPG combining:

- Diablo-style real-time combat
- Genshin-style party swapping
- Diablo II and Grim Dawn-inspired loot
- Buildcraft centered on team synergy

## Core Design Pillars

1. Responsive combat
2. Meaningful party composition
3. Loot that changes playstyle
4. Clear visual readability
5. Playable after every development milestone

## Current Scope

The current goal is a small combat prototype, not a complete game.

The prototype must prove:

- Movement feels responsive
- Character swapping feels tactical
- Party abilities interact in useful ways
- Loot can modify team rotations
